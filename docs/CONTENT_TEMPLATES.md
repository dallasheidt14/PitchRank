## Tone Guidelines
- **Engaging:** Use emojis, create excitement
- **Analytical:** Include the "why" not just the "what"
- **Accurate:** Double-check team names and stats
- **Actionable:** Always link to full data

## Platform-Specific Notes
- **Telegram:** Markdown works, tables don't
- **Discord:** Markdown works, use `**bold**` for emphasis
- **Instagram:** No links in captions, use bio link
- **Twitter/X:** 280 char limit, thread for more
