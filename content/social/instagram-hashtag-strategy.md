## Hashtag Rules

1. **Always include** `#pitchrank` and `#powerscore` — build brand recognition
2. **Use 15-20 hashtags** per post (sweet spot for reach vs. spam)
3. **Rotate sets** — don't use the exact same hashtags every post (Instagram may flag as spam)
4. **Place hashtags** in the first comment, not the caption (cleaner look)
5. **Add 1-2 club-specific hashtags** when featuring a team (e.g., `#surfsc`, `#solarisfc`)
6. **Track performance** via Instagram Insights → see which hashtags drive the most reach
7. **Update monthly** — remove underperformers, add trending tags

## Hashtags to AVOID
- `#follow4follow`, `#like4like`, `#instagood` — irrelevant, attracts bots
- `#ad`, `#sponsored` — unless actually sponsored
- Anything with fewer than 1K posts (too niche, no search volume)
- Competitor brand names as hashtags (bad look)

## Growth Milestones & Hashtag Evolution

### Phase 1 (0-1K followers): Focus on niche
- Emphasize Tier 2-3 (niche soccer hashtags)
- Use smaller, less competitive tags where you can rank
- Goal: become discoverable in youth soccer community

### Phase 2 (1K-5K followers): Expand reach
- Add more Tier 5 (broader soccer tags)
- Start using trending hashtags when relevant
- Create content for specific hashtag communities

### Phase 3 (5K+ followers): Brand building
- `#pitchrank` should start appearing in others' posts
- Encourage clubs to use `#pitchrank` when sharing their ranking
- Consider branded hashtag campaign: `#WhatsYourPowerScore`
