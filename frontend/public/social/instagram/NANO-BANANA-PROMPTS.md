# Nano Banana Design Prompts for PitchRank Instagram

Use these prompts in Nano Banana (or Midjourney/DALL-E) to generate alternative versions of each post. Compare with the SVG versions included in this folder.

**Brand Guidelines for all prompts:**
- Primary Color: Forest Green `#0B5345`
- Dark Green: `#052E27`
- Accent: Electric Yellow `#F4D03F`
- Fonts: Oswald (headlines), DM Sans (body)
- Style: Modern sports data visualization, athletic editorial
- Always include the diagonal yellow slash accent element
- Logo: "PITCHRANK" with "RANK" in yellow

---

## Tips for Nano Banana

1. **Add `--ar 1:1`** to force square aspect ratio for Instagram posts
2. **Add `--style raw`** if available for less stylization
3. **Try variations** by adding: "photorealistic", "3D render", "flat design", or "sports broadcast overlay"
4. **For text-heavy designs**, AI generators struggle — you may need to add text overlays in Canva/Figma after generation
5. **Generate the background/visual elements** with Nano Banana, then overlay text and brand elements manually for best results
