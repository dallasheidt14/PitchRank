# Logo Assets

This directory should contain the following logo files:

- `favicon.ico` - Browser tab icon
- `pitchrank-logo-dark.png` - Dark mode logo (for header)
- `pitchrank-logo-light.png` - Light mode logo (for header)
- `pitchrank-wordmark.svg` - Full-width branding for landing page
- `pitchrank-symbol.svg` - Symbol logo (for header in light mode)

## Usage

- **Favicon**: Referenced in `app/layout.tsx` for browser tab icon
- **Header Logos**: Used in `components/Navigation.tsx` with automatic theme switching
- **Wordmark**: Displayed on landing page (`app/page.tsx`)

## Note

Copy your logo files from `PitchRank/logos/` to this directory before building/deploying.
