# Logo Assets

This directory contains the following logo files:

- `favicon.ico` - Browser tab icon
- `pitchrank-logo-white.svg` - Light mode logo (for header)
- `pitchrank-logo-black.svg` - Dark mode logo (for header)
- `pitchrank-wordmark.svg` - Full-width branding for landing page
- `pitchrank-symbol.svg` - Symbol logo

## Usage

- **Favicon**: Referenced in `app/layout.tsx` for browser tab icon
- **Header Logos**: Used in `components/Navigation.tsx` with automatic theme switching (SVG format for smaller bundle size)
- **Wordmark**: Displayed on landing page and social sharing

## Note

Copy your logo files from `PitchRank/logos/` to this directory before building/deploying.
